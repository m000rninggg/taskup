# TaskUp

TaskUp - веб-приложение для организации командной работы над проектами. Система помогает создавать команды и проекты, вести задачи в таблице и Kanban-доске, хранить проектную документацию, обсуждать задачи в комментариях и отслеживать состояние выполнения работ.

## Основные возможности

- регистрация, авторизация и управление профилем пользователя;
- создание команд и управление участниками;
- создание проектов внутри команд;
- управление задачами, сроками, статусами и исполнителями;
- представление задач в виде таблицы и Kanban-доски;
- комментарии к задачам;
- ведение проектной документации;
- аналитика по статусам задач;
- настройки проекта и управление доступом.

## Технологии

- PHP 8.2
- Laravel 12
- Laravel Breeze
- Vite
- Alpine.js
- SQLite
- PHPUnit

## Установка

```bash
composer install
npm install
cp .env.example .env
php artisan key:generate
php artisan migrate
npm run build
```

Для запуска локального сервера:

```bash
php artisan serve
```

Для разработки фронтенда:

```bash
npm run dev
```

## Проверка проекта

```bash
vendor/bin/pint --test
php artisan test
npm run build
```
